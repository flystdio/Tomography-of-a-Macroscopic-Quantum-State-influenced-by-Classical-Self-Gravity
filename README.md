# Tomography-of-a-Macroscopic-Quantum-State-influenced-by-Classical-Self-Gravity

Plotting data for the paper "Tomography of a Macroscopic Quantum State influenced by Classical Self-Gravity".

## Data

- `Data/SN_ellipse_two_angle_sets/`: numerical data for the SN/QG covariance ellipse comparison with `T_tomo = 3e-4 K`, `Omega_q / 2pi = 0.1 Hz`, and free evolution disabled.
- `Data/determinate/`: numerical data for the SN reconstructed determinant parameter-space plot over `Omega_q / 2pi = 0.01...10 Hz` and `T = 1e-5...1e-2 K`.
- `Data/cov_error_components_Tcompare/`: numerical data for the covariance error component comparison over `Omega_q / 2pi = 0.01...10 Hz` at `T_tomo = 1e-4 K` and `1e-3 K`.
- `Data/H2_contour_matchedT_manualTime_t0/`: numerical data for the matched-temperature manual-time tomography contour plot at `t_eval = 0`.
- `Data/noise_components_vs_omega/`: numerical data for the noise determinant versus measurement-strength plots at `T_tomo = 1e-4 K` and `1e-3 K`.

Each dataset directory contains a local `README.md`, machine-readable metadata, CSV data tables, and a PDF copy of the corresponding figure when available.
