# Tomography-of-a-Macroscopic-Quantum-State-influenced-by-Classical-Self-Gravity

Plotting data for the paper "Tomography of a Macroscopic Quantum State influenced by Classical Self-Gravity".

## Data

- `Data/tomography_state_fidelity/SN_ellipse_two_angle_sets_T0p0003_Oq0p1_nofree_t0/`: numerical data for the SN/QG covariance ellipse comparison with `T_tomo = 3e-4 K`, `Omega_q / 2pi = 0.1 Hz`, and free evolution disabled.

Each dataset directory contains a local `README.md`, machine-readable metadata, CSV data tables, and a PDF copy of the corresponding figure when available.
